# Fine-tuning-BERT-for-STS

In this project, we have fine tuned 3 models for the regression task of calculating Semantic Text Similarity between 2 input sentences.

The models used are:
1. XLM-ROBERTa
2. mBERT
3. DistilBERT

For implementation details, dataset details, results and conclusions please refer to the [Report](Report.pdf)
